1. Command: nmap -sn 10.0.2.15/24
=====================================================================

Step:
- I performed a host discovery scan on the 10.0.2.15/24 network.

Purpose:
- Checks which machines are online.
- Does not scan ports (Ping Scan).

Observation:

- No port details were shown.

Results:

  Example Output:
    Nmap scan report for 10.0.2.2

    Nmap scan report for 10.0.2.3
    Host is up.
    Nmap scan report for 10.0.2.15

2. Command: nmap 10.0.2.2
=====================================================================

Step:
- Scanned a single machine using Nmap default scan.

Purpose:
- Performs a SYN scan (if privileged).
- Shows open ports and basic service guesses.

Observation:
- Returned standard ports such as TCP (445) and  TCP (135) depending on VM setup.

Result:
  Example Output:
    PORT     STATE SERVICE
    135/tcp   open  msrpc
    445/tcp   open  microsoft-ds
    3306/tcp  open  mysql
    3389/tcp  open ms-wbt-server

4. Command: nmap -v -p21 -sV -T4 10.0.2.2
=====================================================================

Step:
- Ran a verbose service version scan against port 21 (FTP).

Purpose:
- Detects the service and version running on port 21.
- -T4 increases speed.
- -sV reveals service versions.

Observation:
- The FTP port is being filtered by a firewall.

Result:
  Example Output:
    PORT   STATE     SERVICE VERSION
    21/tcp filtered  ftp     

Screenshot Reference:
- /nmap-labs/screenshots/04-service-detection.png

6. Command: nmap -A -p135,445 10.0.2.2
=====================================================================

Step:
- Scanned SMB (Server Message Block) ports with aggressive mode.

Purpose:
- Collects detailed SMB information.
- Useful for Windows/SMB enumeration.

Observation:
- Showed Microsoft Windows RPC details depending on host configuration and host script results.

Result:
  Example Output:
    PORT    STATE SERVICE       VERSION
    135/tcp open  msrpc         Microsoft Windows RPC
    445/tcp open  microsoft-ds?
    Service Info: OS: Windows; CPE: cpe:/o:microsoft:windows

Screenshot Reference:
- /nmap-labs/screenshots/06-smb-scan.png
